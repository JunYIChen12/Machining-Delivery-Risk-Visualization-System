# Machining Delivery Risk Visualization System

机加工生产事实可视化产品原型仓库。

本仓库用于沉淀“液压快速换模系统”在机加工多订单交叉生产场景下的产品设计迭代、原型 Demo 和产品文档。

## 当前产品定义

- 产品族：液压快速换模系统
- 订单型号：例如 25T、30T、10T
- 固定半成品结构：销轴、缸体、活塞、压板、基座
- 固定工艺路线：绑定在半成品模板上
- 订单变化项：每个半成品的规格、目标数量、加工进度、报工状态

## 第一阶段产品边界

- 展示可信报工链路形成的生产事实
- 按“半成品名称 + 规格”形成产能池
- 支持订单向下追溯、工序向上追溯
- 不读取不可信库存
- 不自动判断订单一定能否交付
- 不自动判断半成品最终服务哪张销售订单

## 目录说明

- `docs/`：产品原型文档，含 Markdown 版最新文档
- `prototype/`：当前最新可运行 Demo
- `versions/`：历次产品设计版本沉淀

## 当前 Demo

打开 `prototype/index.html` 即可查看当前版本。

当前 Demo 的重点：

- 固定 BOM 规格流转地图
- 半成品规格产能池
- 订单相关加工进度
- 工序热力图
- 自上而下、自下而上的追溯表达
